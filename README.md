# 💊 药企 AI 工具集

四个面向药企场景的 Python 项目。探索了 RAG 技术在临床文档问答中的应用（示例脚本）。

| 项目 | 说明 | 文档 |
|---|---|---|
| 临床试验方案信息提取器 | 从 Word 文档自动提取注册号、日期、终点 | [README](README_extract_protocol.md) |
| 数据快速探索工具 | Streamlit Web 应用，上传即画散点图 | [README](README_streamlit_app.md) |
| 轻量 RAG 检索 | LangChain 向量检索 + 模拟 LLM 问答 | [README](README_rag_search.md) |
| 药物应答率分析 | 按药物分组计算应答率，Plotly 柱状图 | [README](README_plot_response.md) |

🌐 在线体验：[数据快速探索工具](https://beigene-ai-intern-portfolio-bv3qvff4bsnpspg9puec2b.streamlit.app/)

## 环境

```bash
pip install -r requirements.txt
```

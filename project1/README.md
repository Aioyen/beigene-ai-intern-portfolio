# 临床试验方案信息提取器

从 Word 文档中自动提取临床试验关键信息。

## 功能

读取 `.docx` 或 `.doc` 文件，自动提取：

- 临床试验注册号（NCT / CTR 格式）
- 日期（YYYY-MM-DD）
- 主要终点 / 研究目的描述

结果保存为 `extracted_info.csv`。

## 快速开始

```bash
# 自动扫描目录下的 Word 文件
python extract_protocol.py

# 指定文件
python extract_protocol.py "CTR20262087详细信息.docx"
```

## 示例输出

| 类型 | 提取内容 |
|---|---|
| 注册号 | CTR20262087 |
| 日期 | 2026-02-28 |
| 日期 | 2026-03-13 |
| 日期 | 2026-05-28 |
| 主要终点/目的 | 评价曲拉西利相比安慰剂对局限期小细胞肺癌... |

## 技术栈

- `python-docx` — 解析 Word 段落和表格
- `re` — 正则匹配注册号、日期、中文关键词
- `LibreOffice` — `.doc` 自动转 `.docx`（可选）

## 正则规则

| 目标 | 正则 |
|---|---|
| 注册号 | `(NCT\|CTR)\d{6,8}` |
| 日期 | `\d{4}-\d{2}-\d{2}` |
| 主要终点 | `主要终点/主要目的/主要评价指标 + 后跟句子` |

## 依赖

```bash
pip install python-docx
```
